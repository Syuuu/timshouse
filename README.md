# JLPT N2 小雨学习 Web App

使用 Next.js 14（pages 路由）+ React 编写的轻量 N2 学习小站，包含每日任务、卡片式学习与小测功能。

## 功能速览
- 今日任务：新单词、复习单词、新语法、复习语法数量与完成进度。
- 连续天数：展示连续学习天数与段位称号，激励保持节奏。
- 学习卡片：翻转查看中文释义与例句，按记忆程度设定下次复习间隔（1/3/7 天，代码中可修改）。
- 小测试：四选一单词题、语法选择题，自动记录正确率与历史表现。
- 数据持久化：使用浏览器 localStorage 记录复习安排和学习历史。

## 运行
```bash
npm install
npm run dev
```

## 构建与部署到 Vercel
1. 将代码推送到 GitHub 仓库。
2. 在 Vercel 导入仓库，框架选择 Next.js，保持默认构建命令 `npm run build` 与启动命令 `npm start`。
3. 部署完成后即可分享小雨的专属学习入口。
